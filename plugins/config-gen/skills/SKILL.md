# Configuration Generator Skill

## Overview

This skill guides Claude Code to operate the `cfg_gen` tool for querying configuration table information and updating configuration tables in game development projects.

## Purpose

Provide specialized guidance for:
- Querying configuration table data
- Generating and updating configuration files
- Managing game configuration schemas
- Validating configuration table structures

## Core Capabilities

### Configuration Query Operations
- Query specific configuration table entries
- Search configuration data by criteria
- Extract configuration information for analysis
- Generate configuration reports

### Configuration Update Operations
- Update existing configuration entries
- Add new configuration records
- Validate configuration changes
- Generate configuration diffs

### Schema Management
- Define configuration table schemas
- Validate configuration structures
- Generate configuration templates
- Manage configuration versioning

## Trigger Scenarios

This skill should be triggered when:
- User requests to query configuration table data
- Configuration updates are needed for game development
- Configuration validation or analysis is required
- Configuration schema changes are proposed

## Workflow Guidelines

### Query Operations
1. Identify the target configuration table
2. Specify query criteria (IDs, fields, conditions)
3. Execute query using cfg_gen tool
4. Present results in structured format

### Update Operations
1. Review current configuration state
2. Validate proposed changes
3. Execute updates using cfg_gen tool
4. Verify changes and provide confirmation

### Schema Operations
1. Analyze configuration structure requirements
2. Define or modify schema definitions
3. Validate schema compatibility
4. Generate configuration templates

## Notes

- Always validate configuration changes before applying
- Maintain configuration consistency across tables
- Follow project-specific configuration conventions
- Document configuration changes for traceability
