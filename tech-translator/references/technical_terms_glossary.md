## 翻译原则

1. **首次出现规则**：技术术语首次出现时使用中文翻译，英文原词放在括号中
2. **一致性原则**：同一文档中相同的术语使用相同的翻译
3. **保留原则**：专有名词、公司名、产品名保持原样
4. **可读性原则**：在保持技术准确性的同时确保翻译流畅自然

## 特殊处理

### 保留原样的术语
- 公司名称：Google, Microsoft, Apple, Amazon
- 产品名称：Windows, macOS, iOS, Android
- 项目名称：React, Vue, Angular
- 标准名称：HTTP, TCP, UDP

### 需要翻译的术语
- 通用计算机术语：computer → 计算机
- 基础数学术语：algorithm → 算法
- 日常技术词汇：software → 软件
